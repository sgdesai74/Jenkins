# Jenkins
Key Tools and Technologies used
Jenkins: Automation server used for continuous integration and continuous delivery (CI/CD).
AWS EC2: Virtual machine in the cloud where Jenkins is hosted.
Apache Ant: Build tool integrated with Jenkins for compiling and running Java-based projects.
Git: Version control system used to manage the source code repository.
GitHub: Platform where the source code repository is hosted, and you will fork and clone the repository.
AWS IAM: Identity and Access Management for secure access to the AWS Management Console.
Shell Scripting: Used to install and configure necessary tools like Ant and Git on the EC2 instance.
